Pseudo-Concepts with requires<>

Detailed Description

These type traits check the type properties of objects that act as containers.

Requires for containers.

The _vt methods check that a containers value_type fulfills certain conditions while _st methods check that the objects scalar_type fulfills certain conditions. value_type and scalar_type differ in that value_type is the first level of a container while scalar_type recursivly goes through containers of containers till it comes to a simple type.
